# Teslo API

1. Clone the repository
2. Install dependencies 
```
npm install
```
3. Create a .env file with the .env.template file
4. Run the following command to start the database:
```
docker compose up -d
```
5. Run the following command to start the server:
```
npm run start:dev
```
